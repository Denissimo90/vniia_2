Справка по API

Роуты

  • /factories - GET запрос возвращает список предприятий

  • /factories/:factoryId - GET запрос возвращает более детализированную информацию по конкретному предприятию (:factoryId) и список выпускаемой им продукции
    :factoryId можно посмотреть в предыдущем запроса - (поле id)

  • /factories/:factoryId/:timeKey - GET запрос информацию по конкретному предприятию (:factoryId) и список произведенной в (:timeKey) продукции.
    :timeKey подается в формате YYYY-MM-DD
